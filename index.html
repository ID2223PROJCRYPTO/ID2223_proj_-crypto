Crypto Price Direction Prediction

This project focuses on predicting the 24-hour price direction of Bitcoin (BTC) using machine learning models trained on historical price data and engineered market features. The goal is not exact price forecasting, but binary classification: whether the price will go up or down on the next day.

Task type: Binary classification
Target: BTC price direction (t + 24h)
Features: Price-based indicators & engineered features
---

Method Overview

The pipeline consists of data ingestion, feature engineering, model training, and evaluation. Raw market data is transformed into supervised learning samples using a time-aware labeling strategy to avoid data leakage.

  • Historical BTC market data collection
  • Feature engineering (returns, trends, lag features)
  • Supervised learning model training
  • Evaluation using classification metrics

Conclusion

This project demonstrates how machine learning techniques can be applied to financial time-series classification problems. Although market prediction remains inherently uncertain, the results indicate that engineered features combined with supervised learning models can extract non-trivial predictive patterns.

Future improvements may include incorporating additional external signals, model ensembling, and more advanced temporal architectures.
